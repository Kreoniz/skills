# Skills

Personal Codex skills and mirrored third-party skills.

## Included

- `caveman`
- `grill-me`

## Install

```bash
git clone https://github.com/Kreoniz/skills.git
mkdir -p ~/.codex/skills
cp -R skills/skills/* ~/.codex/skills/
```

Restart Codex after installing or updating skills.
